I am a Robotics Engineer working at Systemantics India Pvt. Ltd. I am experienced in embedded programming, robot control, motion planning, robot kinematics & dynamics and mechanical design. Projects

Robot Mechanics

July 2018 - Present
  • Mathematically modelled a novel hybrid (serial plus parallel) architecture with six degrees of freedom industrial robot.
  • Formulated a computationally efficient line search algorithm to solve the robot inverse kinematics problem. Also an analytical forward kinematics solution was obtained.
  • Identified all robot singularities and implemented a singularity tracking and avoidance algorithm.
  • Designed workspace for the hybrid robot that could compete with those of standard industrial serial models.

Robot Control

July 2018 - Present
  • Implemented and tuned robust discrete-time joint level controllers in Embedded C for the robot axes.
  • Modelled and experimentally identified parameters of the joint friction and dynamic model.
  • Derived the inverse dynamics analytical model of the hybrid robot and implemented for feedforward control.
  • Worked on the control of flexible joints using output measurements from MEMS triaxial accelerometer.

Motion Planning

July 2018 - Present
  • Implemented a task space jerk limited trajectory generation algorithm for point to point motions.
  • Tuned these motion parameters for the robot with considerations of robot bandwidth and actuator capabilities.
  • Explored various path blending strategies for position and orientation and devised algorithms that satisfied practical real-time computation limitations.
  • Worked on higher continuity quaternion splines that balance computational complexity with desired smoothness.

Robot Calibration

December 2016 - May 2017
  • Developed a comprehensive calibration model for a 4 degree of freedom planar parallel manipulator that took into account all possible inaccuracies arising from manufacturing and assembly.
  • Experimentally obtained task-space calibration data via a FARO laser tracker.
  • Implemented a stochastic gradient descent optimization algorithm to obtain calibrated model parameters resulting in improved robot accuracy.
  • Conducted a sensitivity analysis to identify critical parameters and help reduce the complex model to an analytically solvable one
